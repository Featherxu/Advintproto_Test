THE BRIEF

In this project we will work with collecting and transmitting data, as well as representing it in the form of a dashboard.

INSPIRATION

I got the inspiration when i was tidying up my documents and saw my Artcenter offer by chance.

I love patterns, and these dots reminds me the artwork from Yayoi Kusama and her bold color choice. I think it's interesting to explore it in processing and the controllers.  if users could play with it, it would be very fascinating. That's how I got my inspiration and decide to work on this topic.

ENVISION
CONCEPT

HOW VISION CHANGED

I'm not only curious about it, but also it's necessary for me to know if she went to eat or poop so I can go clean up for her on time. At first I was trying to put the particle board on a cat harness and detect where she goes, but turns out that my cat can't wear harness at all. So i decide to put the sensor beside her food and litter box instead to do the same thing by collecting different data.

PRESSURE

This is a force sensitive resistor with a round, 0.5" diameter, sensing area. It can detect if there's pressure on it or not. I put it under my cat's food bowl and with a card board covering.

MOTION

The IR sensor I used could detect motion. It can tell if something is moving or not. I place it in the path beside Zero's litter box so it knows when she pass by.

Data Visualization

HOW I SHOW THE DATA

Besides from using the right sensor, another big part of this project is to reflect the data onto my web page. I used P5Js to connect with my particle board and did the graphic on it. I tried with WebGL to make the graphic more impressive.

PROCESS

Setting up the sensors

To beginning with, I set up the 2 sensors i chose on the bread board. It takes me long time to figure out because I wasn't clear about the differences of analog input and digital input.

Connecting to Particle IO

The next step is to connect my sensors with the particle.io so I can know if the data is going through or not. I noticed the delay time is very crucial for the data coming in.

Usability Testing

After I'm done with the prototype, I test it with my cat for iterations. For example, the force sensor was on the bread board but after testing i decide to use extra wires to extend it

Start Web Page

Now I get the data from the sensors detecting the environment, I want to make it shown on a web page with WebGL for data visualization.

Future Developments

Planning for the future steps, I want to make a box or holder for the sensors to make sure my cat won't destroy it. I also want to add more sensor to better detect more of my cat's daily behavior like where she goes or when she sleeps.
